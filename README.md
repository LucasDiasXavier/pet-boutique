# 🐾 Pet Boutique

## 📸 Visão Geral

A **Pet Boutique** é uma landing page criada para uma petshop especializada em cães e gatos. O projeto apresenta os principais serviços oferecidos, marcas parceiras, localização via Google Maps e um botão de contato direto pelo WhatsApp — tudo com um visual elegante e moderno.

---

## 🚀 Funcionalidades

- **Banner principal** com imagem de fundo, logo e chamada para ação
- **Botão de WhatsApp** com link direto para atendimento ao cliente
- **Seção de Serviços** com lista detalhada e imagens ilustrativas de cada serviço
- **Seção Sobre** com destaque para a identidade da marca e links para redes sociais
- **Vitrine de Marcas Parceiras** com grid de logos de 12 marcas
- **Mapa interativo** do Google Maps com a localização da loja
- **Rodapé** com informações de copyright

## 🎨 Design & Identidade Visual

O projeto utiliza uma paleta que transmite elegância e sofisticação, alinhada ao posicionamento premium da marca:

| Cor | Hex | Uso |
|---|---|---|
| Roxo escuro | `#3c184e` | Cor principal — textos, fundos, botões |
| Branco translúcido | `#ffffffc4` | Textos sobre imagem no banner |
| Branco puro | `#ffffff` | Textos em fundos escuros |
| Sombra escura | `#0000005c` | Box-shadow nas imagens de serviços |

**Tipografia:** `sans-serif` nativa do sistema — limpa e legível em todos os dispositivos.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e organização do conteúdo
- **CSS3** — Estilização, layout, sombras e responsividade básica
- **Google Maps Embed API** — Mapa interativo na seção de localização
- **WhatsApp API (wa.me)** — Link direto para atendimento via WhatsApp

## ⚙️ Como Executar o Projeto

Nenhuma dependência ou build necessário. Basta clonar e abrir no navegador.

```bash
# Clone o repositório
git clone https://github.com/LucasDiasXavier/pet-boutique.git

# Acesse a pasta do projeto
cd pet-boutique

# Abra no navegador
# Dê duplo clique em index.html, ou use a extensão Live Server no VS Code
```

> 💡 **Dica:** Use a extensão **Live Server** do VS Code para uma melhor experiência de desenvolvimento com recarregamento automático.
